# Ai-project-1

Project 1: Prediction of Deaths on the Titanic:
This project is an introduction to machine learning, using a regression model I managed to predict who would’ve died in the crash of the titanic.
It has taught me how to format datasets, find correlations between the parameters and the final result by graphing them. 
I have also learned that tweaking the parameters of a model can improve the performance but can also decrease it if the value of the parameters goes past the optimum point

Lib:
Catboost
Sklearn
pandasw
